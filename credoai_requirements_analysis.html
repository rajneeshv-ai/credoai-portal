DPDPA 2023 (India)

Article 6
Purpose Limitation
AI may only process data for the exact purpose declared at consent time. Requirement: OPA gates every request against purpose-binding JWT claims.
Article 8
Data Minimisation
LLM must never receive raw PII. Requirement: Microsoft Presidio scrubbing is mandatory at the .NET ingestion boundary before the payload hits Kafka.
Article 12
Right to Erasure
Deletion propagation within 72 hours. Requirement: Neo4j graph nodes must support cascading tombstone updates without corrupting existing AI trace histories.
Article 17
Data Residency
Indian resident data cannot cross borders. Requirement: Physical infrastructure locked to Azure India Central with Private Endpoints (no public egress).

RBI IT Governance 2024

Sec 4.3
Incident Logging
Decisions affecting customer accounts require full audit reconstruction. Requirement: W3C `traceparent` headers across C#, Kafka, and Python.
Sec 6
Mandatory Human Oversight
Automated AI in finance requires a human kill-switch. Requirement: GraphRAG Confidence Gate < 0.75 triggers deterministic LangGraph HITL checkpoint.

Intelligent Orchestration

Multi-hop Graph Retrieval
System must resolve rules spanning multiple connected documents. Flat RAG is structurally rejected. Requirement: Neo4j AuraDB integrated with Azure AI Search via GraphRAG.
Cyclic Execution State
Banking workflows require retries, tool calls, and pausing for human review. Requirement: Python runtime must use LangGraph (state machine) rather than LangChain (linear pipeline).

Distributed Reliability

Zero State Drift Guarantee
The SQL business ledger and the Kafka event stream must never disagree on application state. Requirement: Atomic Transactional Outbox pattern implemented in EF Core.
Eventual Consistency Rollback
If Python inference fails permanently, locked DB funds must be released. Requirement: Distributed Saga Compensation orchestrated by .NET on Kafka DLQ events.

AI Quality Metrics (RAGAS)

Faithfulness ≥ 0.90
AI must not hallucinate facts outside the retrieved RBI context.
Answer Relevance ≥ 0.85
AI must directly answer the user's prompt without tangential drift.
Context Precision ≥ 0.80
The GraphRAG retrieval must rank the most relevant clauses first.
Context Recall ≥ 0.85
The retrieval must fetch all necessary interconnected rules.

STRIDE + AI Threat Mitigations

Prompt Injection bypassing Governance
Threat: User tricks LLM into leaking data.
Mitigation: OPA policy engine evaluates JWTs *outside* the LLM. NeMo Guardrails block adversarial intents before LangGraph execution.
Cross-Tenant Graph Leakage
Threat: Cypher queries returning another bank's rules.
Mitigation: Mandatory `Tenant_ID` label isolation applied to all Neo4j queries at the repository pattern layer.
Confidence Manipulation
Threat: Prompt forces LLM to declare high confidence on a hallucination.
Mitigation: Confidence is calculated mathematically via Subgraph Density, immune to prompt engineering.
