Architectural Insight: We intentionally reject Two-Phase Commit (2PC) across polyglot boundaries because AI inference is slow and probabilistic. Instead, we use Eventual Consistency. If the Python agent fails 3 times, Kafka routes the message to the DLQ, triggering a .NET Saga Compensator to cleanly rollback the initial PostgreSQL ledger entry. Zero state drift.
